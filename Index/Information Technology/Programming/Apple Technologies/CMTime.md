- **Notes**
	```swift
	CMTimeMake(value: 1, timescale: 1) // 1 second
	CMTimeMake(value: 1, timescale: 2) // 0.5 seconds
	CMTimeMake(value: 2000, timescale: 1000) // 2 seconds
	CMTimeMake(value: 2000, timescale: 3000) // 0.6666666 seconds
	```
