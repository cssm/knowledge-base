## Naming

### [[Swift Enums]]

If enum is used for namespacing - use plural name e.g.

```swift
enum Deeplinks {
	struct Universal {}
	struct URLSchema {}
}
```

In other cases use singular

```swift
enum State {
	case idle
	case loading
}
```
