 - *Snippets* 
	- Rename file on case-insensitive systems (`UNIX`-like)
		- `git mv readme.md README.md`
	- Remove from remote everything that not match with local    
		- `git push --force-with-lease origin $BRANCH`
	- Remove file only from git
		 - `git rm --cached <file>`
